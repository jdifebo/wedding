# Our wedding website
I figured it was pretty silly to have someone else make one when it's pretty easy to throw one together myself

## Built with:
    * Backend
        * Java 8 & Kotlin
        * mySQL
        * Spring Boot
        * Spring Web MVC
        * Spring Data JPA & Hibernate
    * Frontend
        * Bootstrap 4 (Alpha)
        * jQuery
    * Infrastucture
        * Google Cloud
        * Apache
        * LetsEncrypt SSL Certificate
        * NameCheap for 88 cent domain
